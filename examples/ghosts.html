This demo is a clone of Angry Birds that integrates the Box2dWeb Javascript library with custom rendering in EaselJS. The artwork is stolen from the awesome Seb Lee-Delisle's live coding example of the game "Ghosts and Monsters" using the Corona SDK for buildling iOS and Android apps.

To play: click and drag the head to the left, then relase it to sling it at the blocks
